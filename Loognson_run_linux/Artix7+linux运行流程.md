run_linux文件夹结构

* tftpd32.450 [轻量的[TFTP](https://en.wikipedia.org/wiki/Trivial_File_Transfer_Protocol)服务器]
  * 运行 tftpd32.exe 以搭建TFTP服务器
* SecureCRTPortable [一款支持[SSH](https://www.ssh.com/academy/ssh)的终端仿真程序, 远程登陆Linux/Unix]
  * 运行 SecureCRTPortable.exe 以配置终端
